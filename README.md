# НейроХам 2.0 👿

Дерзкий, матерный, саркастичный Telegram-бот в стиле оригинального НейроХама.

## Фичи
- Ответы без цензуры и фильтров
- Генерация мемов и картинок (Grok Imagine / DALL-E)
- Мини-игры: Камень-ножницы-бумага, Угадай число
- Ежедневные троллинг-задания
- Память контекста разговора
- Режимы хамства

## Как запустить

1. `git clone https://github.com/spoty4600-web/neiroham-bot.git`
2. `cd neiroham-bot`
3. `pip install -r requirements.txt`
4. Создай файл `.env`:
