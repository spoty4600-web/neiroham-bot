# Рейдер

Дерзкий Telegram-бот Рейдер с ИИ и генерацией картинок.

## Запуск

1. `git clone https://github.com/spoty4600-web/neiroham-bot.git`
2. `cd neiroham-bot`
3. `pip install -r requirements.txt`
4. Скопируй `.env.example` в `.env` и вставь свои ключи:
   - `BOT_TOKEN` - получи от [@BotFather](https://t.me/botfather) в Telegram
   - `GROQ_API_KEY` - бесплатный ключ с https://console.groq.com
5. `python main.py`

## Команды

- `/start` - запуск бота
- `/help` - справка по командам
- `/clear` - очистить историю диалога
- `/pic [описание]` - сгенерировать картинку

## Бесплатные ресурсы

- **ИИ для текста**: Groq API (llama-3.3-70b) - БЕСПЛАТНО
- **Генерация картинок**: Pollinations.ai - БЕСПЛАТНО
- **Telegram Bot**: Telegram Bot API - БЕСПЛАТНО
